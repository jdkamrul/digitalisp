# ISP Digital ERP — Development Status

## Project Overview
Full-featured ISP management ERP built in PHP (no framework), SQLite/MySQL, vanilla JS.
Covers billing, network management, GPON, inventory, customer portal, payments, automation, and communication.

---

## ✅ Completed Features

### Core Infrastructure
- Custom MVC router (`app/Core/Router.php`) with prefix, middleware, and param support
- SQLite auto-schema with column migration (`config/database.php` — `ensureSqliteSchema` + `runColumnMigrations`)
- Session-based auth with role support (`AuthMiddleware`, `AuthController`)
- Global helper functions: `base_url`, `sanitize`, `json_response`, `redirect`, `formatMoney`
- Dark/light theme toggle with `localStorage` persistence
- Collapsible sidebar (desktop icon-rail + mobile slide) with `localStorage` state
- Global customer search (header, keyboard shortcut ⌘K)
- Responsive layout (`views/layouts/main.php`)

### Customer Management
- Customer list with search, filter by status/package/zone, pagination, CSV/PDF export
- Create / Edit / View customer profiles
- Bulk create, bulk update, bulk suspend, bulk reconnect, bulk delete
- Customer status log (`customer_status_log` table)
- PPPoE user auto-creation on new customer
- Auto installation work order on new connection
- Auto-payment subscription display on customer profile
- Customer portal (separate auth, dashboard, billing, profile, support, usage)

### Billing & Invoicing
- Invoice generation (manual + pro-rata for new connections)
- Invoice detail view with payment history

- Receipt printing
- Cashbook view
- Pay via PipraPay button (external API)
- Pay via Self-Hosted PipraPay button

### Self-Hosted PipraPay Payment System ✅ Fully Complete

## Architecture Summary

```
public/index.php
    └── config/app.php (autoloader, helpers, session)
    └── config/database.php (PDO, SQLite auto-schema)
    └── routes/web.php → Router::dispatch()
        ├── app/Controllers/*.php
        │   ├── AuthController
        │   ├── CustomerController
        │   ├── BillingController
        │   ├── NetworkController
        │   ├── ModuleControllers (Gpon, Inventory, Reseller, Reports, Finance, Settings, WorkOrder)
        │   ├── PipraPayController
        │   ├── SelfHostedPipraPayController
        │   ├── CommunicationController
        │   └── AutomationController
        ├── app/Services/*.php
        │   ├── MikroTikService (RouterOS API)
        │   ├── RadiusService (FreeRADIUS DB)
        │   ├── SmsService (bulk SMS, templates)
        │   ├── PipraPayService (external API)
        │   ├── SelfHostedPipraPayService (local payment)
        │   ├── AutomationService (billing jobs)
        │   ├── SnmpOltService (GPON SNMP)
        │   └── TelnetOltService (GPON Telnet)
        └── views/layouts/main.php → views/**/*.php
    └── routes/portal.php → CustomerPortal controllers
```
